Translation Settings

Enable automatic detection of the source language based on the selected text. When enabled, the extension will automatically determine the language of the text, saving you the effort of manually selecting it.

Choose the specific source language for translation. Select the language from the drop-down menu to set it as the source language for all translations performed by the extension.

Select the desired target language for translation. Choose the language from the drop-down menu to specify the language to which the selected text will be translated.

Select the translation API to be used for language translation. Choose the desired translation service from the dropdown menu to determine the underlying technology that powers the translation functionality of the extension. Selecting the appropriate API ensures accurate and reliable translations based on your preferences and requirements.

Select how translations are displayed. Choose between tooltip or pop-up window for the translation format. Tooltip shows a small box when hovering, while pop-up window displays the translation in a separate window. Customize your translation experience with ease.

LingoScan Settings

Activate lemmatization to delve deeper into word analysis. Lemmatization converts words to their base form, revealing essential linguistic insights. Explore language with enhanced clarity and understanding. Experience the magic of lemmatization today.

Choose the API for performing word breakdown and analysis. Select between Merriam-Webster Dictionary API and Oxford Dictionaries API to enhance your language comprehension and understanding. Get in-depth definitions, synonyms, and usage examples for better context and clarity.

Select the preferred display option for the word breakdown and analysis feature. Choose between displaying the breakdown as a tooltip or a pop-up window for a seamless and convenient user experience. Customize the display to suit your preference and make the most out of the LingoScan functionality.
